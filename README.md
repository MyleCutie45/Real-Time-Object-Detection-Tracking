# Real-time Object Tracking with YOLOv8 and SORT

A real-time object tracking system using **YOLOv8** for object detection and **SORT (Simple Online Realtime Tracking)** for multi-object tracking in video streams.

## Features

* Real-time object detection with YOLOv8
* Multi-object tracking using SORT
* Supports webcam and video files
* Fast and lightweight performance
* Bounding boxes with unique tracking IDs

## Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* SORT Algorithm

## Workflow

Video Input → YOLOv8 Detection → SORT Tracking → Real-time Output Display

## Applications

* Surveillance Systems
* Traffic Monitoring
* Autonomous Vehicles
* Smart City Solutions
